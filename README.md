# formatter

A simple command-line utility to format some files using Python and Typer. (.json, .yaml, .xml, .toml, ...)

## Features
- Pretty-print JSON and YAML
- Supports single or multiple files
- Clean output using Rich
- Simple CLI interface

## Installation
```bash
pip install .
```
